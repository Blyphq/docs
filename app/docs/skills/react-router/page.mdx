# Blyp React Router

Install `blyp-react-router` after `blyp-core` when the project uses React Router. This preset keeps agents on the official middleware-based adapter path, stores request-scoped logging in the router `context`, and mounts `clientLogHandler(request)` on the configured ingestion route.

## When to install it

- the repo uses React Router middleware
- loaders or actions need request-scoped logging through `getLogger(context)`
- browser ingestion is exposed through a React Router route action

## What it covers

- `createLogger()` from `@blyp/core/react-router`
- `middleware`
- `getLogger(context)`
- `clientLogHandler(request)`
- `500` mismatch behavior when the mounted path and configured path diverge
