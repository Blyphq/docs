# Development

## Prerequisites

- [Bun](https://bun.sh) `1.2+` or [Node.js](https://nodejs.org) `18+`
- [TypeScript](https://www.typescriptlang.org) `5+`

## Local setup

```bash
git clone https://github.com/Blyphq/blyp.git
cd blyp
bun install
bun run test
bun run build
bun run type-check
```

## Useful scripts

```bash
bun run test
bun run test:watch
bun run build
bun run type-check
```

## Publishing

The upstream repo publishes to npm from GitHub Actions when a GitHub Release is published. Set `NPM_TOKEN` as a repository secret before using the release workflow.
