# Hono

Import from `blyp-js/hono`.

```ts
import { Hono } from "hono";
import { createLogger } from "blyp-js/hono";

const app = new Hono();

app.use("*", createLogger({
  level: "info",
  customProps: () => ({ framework: "hono" }),
}));

app.get("/hello", (context) => {
  context.get("blypLog").info("hono-route");
  return context.text("ok");
});
```

## Notes

- the request logger is stored in `context.get("blypLog")`
- the middleware handles both success and thrown error flows
- `POST /inngest` is intercepted automatically when client logging is enabled

## Relevant types

```ts
import type {
  HonoLoggerConfig,
  HonoLoggerVariables,
} from "blyp-js/hono";
```
